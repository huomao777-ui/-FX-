## 三、模板 JSON 结构说明

每个模板包含以下字段：

```json
{
  "id": "NEWS_分类_序号",          // 唯一标识
  "enabled": true,                  // 是否启用
  "category": "分类名",             // 6 类之一
  "has_image": true/false,          // 是否有图
  "image_ids": ["IMG_xxx"],         // 图片编号列表（有图时填写）
  "regions": ["漂亮国", "全球"],    // 适用地区（动态占位符）
  "headline_template": "...",       // 主标题模板
  "abstract_template": "...",       // 摘要（一句话点出"发生了什么+对市场的影响"，与标题互补不重复）
  "summary_template": "...",       // 现象总结
  "trend_outlook": "...",          // 趋势预测
  "analysis_tail": "...",          // 正文末尾提示
  "body_template": "...",          // 三段拼接（直接引用上面三个）
  "default_bias": "偏强",          // 默认方向
  "bias_options": ["偏强", "走稳"], // 可选方向
  "default_strength": "明显",       // 默认强度
  "strength_options": ["普通","明显"], // 可选强度
  "reverse_logic_note": "...",     // 反向逻辑说明
  "parameter_hints": ["xxx"],      // 底层接口参数（系统用）
  "tags": ["标签1", "标签2"],       // 备注标签
  "developer_bucket": "xxx"        // 开发者分区（系统用）
}
```

### 字段使用说明

- **regions** 是动态占位符：`{region}` 在游戏中替换为具体地区，`{time_text}` 替换为时间
- **图片** 用编号引用（`IMG_GENERIC_xxx`），**不写路径**
- **方向用语** 只能用：偏强/偏弱/承压/走稳/震荡/分化/放大/收敛/回暖/降温/回吐/修复/拉锯/偏移/等待
- **不允许直接写 "涨" 或 "跌"**
- **analysis_tail** 要把系统参数转成玩家能读懂的市场语言，不暴露参数名

## 四、图片资产系统

文件：`news_image_assets.json`

```json
{
  "images": [
    {
      "id": "IMG_EXCLUSIVE_TRADE_001",
      "file": "res://界面/场景/新闻/美术资源/资讯图/关税战_001.png",
      "type": "exclusive",        // exclusive=专属, generic=泛用
      "category": "贸易与产业",
      "topics": ["关税", "贸易摩擦"]
    }
  ]
}
```

- 3 张专属图（重大事件用）：关税战、加息降息、首相辞职
- 19 张泛用图（各分类可复用），分布：货币政策5、市场预期3、盘面异动3、经济数据2、时政风险2、贸易与产业4
- **模板里只写 `image_ids`**，不写路径，代码层通过 `news_image_assets.json` 解析实际文件路径

## 五、写入规范（代码层需注意）

1. **三段合成正文**：`summary_template + trend_outlook + analysis_tail`
2. **动态替换**：`{region}` → 具体地区名，`{time_text}` → 时间描述
3. **图片选取逻辑**：
   - 优先用 exclusive 类图片（对应类别且匹配 topics）
   - 否则用 generic 类图片（对应类别即可）
   - 可以随机从多个 `image_ids` 中选一张
4. **bias 与强度**：
   - 同类新闻可轮换 bias 和 strength
   - `bias_options` 和 `strength_options` 提供可选范围
   - 反向解读必须有 `reverse_logic_note` 中的经济学理由支撑

## 八、6 类新闻的架构定位

给代码层参考：

- **时政风险** → 触发避险/风险偏好切换，短线冲击型，`developer_bucket: SHORT_SHOCK`
- **货币政策** → 改变利率预期，中长期锚定型，`developer_bucket: LONG_ANCHOR`
- **经济数据** → 修正基本面判断，中期间接型，`developer_bucket: MID_REPRICING`
- **市场预期** → 反映情绪集中/分散，引导方向试探，`developer_bucket: MID_REPRICING`
- **贸易与产业** → 影响出口/增长预期，中长期锚定型，`developer_bucket: LONG_ANCHOR`
- **盘面异动** → 纯短线波动，不改变基本面判断，`developer_bucket: FLOW_MICRO`
